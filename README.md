# ROS 2 Workspace

 **Note:** Please install the [Draw.io VS Code extension](https://marketplace.visualstudio.com/items?itemName=hediet.vscode-drawio) to view the flowcharts (`*.drawio` files) included in this project.

 **Tip:** I highly recommend using the [URDF Visualizer extension](https://open-vsx.org/extension/morningfrog/urdf-visualizer). It was an incredibly convenient tool for viewing the 3D model with hot-reload support, saving the hassle of constantly restarting RViz to visualize even slight changes!
